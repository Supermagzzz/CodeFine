Условный оператор

До сих пор наши программы выполняли четко заданную последовательность действий. Но часто требуется, чтобы программа меняла свое поведение в зависимости от разных данных. Предположим, что вы знаете возраст пользователя и, используя это, хотите показать ему определенную информацию. Что делать в таких случаях?

Чтобы решать подобные задачи нам и понадобится тип bool. Напомним, это тип данных, который принимает одно их двух значений true(правда, 1) или false(ложь, 0).

Предположим, у вас есть переменная age, в которой находится возраст пользователя. Вы хотите проверить есть ли пользователю сорок лет.

bool a = (age >= 40);
Если выполнится условие age >= 40, то в переменную a запишется true, в противном случае false.
Делать подобные проверки вы можете с помощью следующих операторов.
> - больше.
< - меньше.
>= - больше либо равно.
<= - меньше либо равно.
== - равно.
Обратите внимание, что для проверки равенства используется два знака равенства подряд. Это так, потому что один знак равно используется для присваивания значения переменной. Использование одного знака равенства вместо двух - частая ошибка у начинающих программистов. Будьте внимательнее и не путайте эти два оператора.

Мы получили переменную, в которой содержится информация о том, больше ли пользователю лет, чем сорок, или нет. Как работать с этой информацией?

Существует контрукция if (условие) {код}, где код внутри фигурных скобок будет выполнен только если условие равно true.

Начиная с этого урока, мы иногда будем приводить только вырезки кода, так как остальная часть программы не представляет интереса. Чтобы код запустился, у вас обязательно должны быть подключены все библиотеки, объявлены все переменные, и код должен находиться внутри фигурных скобок после строки int main()

bool a = (age >= 10);
if (a)
{
cout << "A ";
 }
cout << "B";
Если пользователю меньше 10 лет, то он увидит надпись "B". Все остальные увидят "A B".
Условие про возраст накладывается только на команду cout << "A "; . Так как cout << "B"; не лежит внутри фигурных скобок условия, то команда выполнится в любом случае.

Код, ограниченный фигурными скобками, называется "Блок кода". Если один блок кода находится внутри другого, то для вложенного блока принято делать отступ в один таб или четыре пробела. Так вам будет легче отличить блоки кода друг от друга и найти условия, к которым они принадлежат. Изучите полный код программы для лучшего понимания.
#include <iostream>

using namespace std;

int main()
{
int age = 24;
bool a = (age >= 10);
 if (a)
{
cout << "A";
 }
cout << "B";
 }
Намного проще писать условие сразу внутри скобок: if().
if (age >= 10)
{
cout << "A";
}
cout << "B";
Блоки кода if можно вкладывать друг в друга, комбинируя несколько условий. Пример:
if (age >= 18)
{
if (country == "Russia")
{
cout << "Hi, adult from Russia";
 }
if (country == "America")
{
cout << "Hi, adult from America";
 }
}
Сначала этот код проверяет, что возраст пользователя больше 18 лет. Если это так, то начинается проверка на страну. Для пользователей из России выведется одно сообщение, для пользователей из Америки другое, а для всех остальных не выведется ничего.

Существует также конструкция else(иначе). Синтаксис следующий:
if(условие)
{код}
else
{код}
Блок кода внутри else выполнится, если условие ложно, и не выполнится блок кода внутри if.

Например, так может выглядеть код, проверяющий, можете ли вы купить товар в интернет-магазине. Если денег достаточно, то вы увидете надпись You can buy this item, а в противном случае You don't have enough money.
if (money >= 100)
{
cout << "You can buy this item";
}
else
{
cout << "You don't have enough money";
}
Также вы можете пользоваться конструкцией else if(условие) {код}, с помощью которой можно собирать целые цепочки условий. Эта конструкция проверяет условие только в том случае, если не выполнилось ни одно из условий выше. Давайте рассмотрим на примере:
if (a >= 100)
{
cout << "A";
}
else if (a >= 80)
{
cout << "B";
}
else if (a >= 60)
{
cout << "C";
}
else
{
cout << "D";
}
Первым в цепочке условий обязан быть блок if. Дальше может быть сколько угодно блоков else if (может не быть вообще). Последним можно добавить блок else.

Условия будут проверяться сверху вниз, и выполнится то, которое истинно и идет первым. На этом проверка условий закончится, а код продолжит выполняться с первой строки, идущей после всей текущей цепочки условий. Если нет ни одного подходящего условия, но существует блок else, то выполнится именно он.

Например, если a = 85, то прошлый код выведет B, в отличии от кода ниже, который выведет BC.
if (a >= 100)
{
cout << "A";
}
if (a >= 80)
{
cout << "B";
}
if (a >= 60)
{
cout << "C";
}
Так как в этом коде все блоки if - это отдельные цепочки условий (состоящие из одного условия), то произойдет проверка всех условий. Помните, блок if всегда начинает новую цепочку.

Сложные условия

Часто нужно писать сложные условия, состоящие из нескольких простых (мы рассматривали их выше). Например:

Если количество денег больше либо равно стоимости товара + стоимость доставки и название страны равно "Россия" и возраст больше 18.

Чтобы комбинировать условия, используются операторы (&& - "И") и (|| - "ИЛИ").
Например, пусть нам нужно условие, которое будет проверять, что человек находится в возрасте от 18 до 30 лет. Но оператора от ... до ... у нас нет. Давайте распишем это условие с помощью уже известных нам операторов. Возраст должен быть больше либо равен 18 и меньше либо равен 30. Напишем это:

if (age >= 18 && age <= 30)
Оператор && соединяет два условия словом И. Если вы помните алегбру логики из школьных уроков информатики, то вам будет проще. Можно также пользоваться скобками. Например:
(условие1 && (условие2 || условие3)) Тогда сначала применится "ИЛИ" между вторым и третьим условием, а потом "И" между первым условием и прошлым результатом.

Давайте посмотрим на следующую табличку, которая формализует работу с логическими операторами. Напоминаем, что:

true - правда.

С опытом вам будет проще понимать сложные условия. А сейчас, давайте решим задачу, которую поставили себе в начале этой части статьи.

Если количество денег больше либо равно стоимости товара + стоимость доставки и название страны равно "Россия" и возраст больше 18.

if (money >= itemCost + deliveryCost && country == "Russia" && age > 18)

В C++ у каждой операции существует приоритет. Например, то, что находится в скобках, будет выполняться раньше. Умножение выполняется раньше сложения. Все как в математике. Операции "больше", "равно", "И" и другие, которые мы изучили в этой статье выполняются позже всех математических операций. Операции "И" и "ИЛИ" выполняются позже всех, это позволяет вам писать ваши условия без скобок. Но если вы понимаете, что в вашей программе есть ошибка, то рекомендуем проверить правильность расстановки скобок в сложных условиях.

Еще один логический оператор - "!" ("НЕ").

Если у вас есть условие, что возраст больше 20 (age > 20), то поставив перед условием восклицательный знак, вы как будто меняете его на обратное. Возраст не больше 20 !(age > 20). У оператора равно есть версия не равно !=. Например x != 5.
