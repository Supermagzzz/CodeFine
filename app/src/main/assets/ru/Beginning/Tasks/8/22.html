B. Разворот (10 б.)
Вам дан массив из n чисел. Выведите его элементы в обратном порядке.
Описание входных данных:
В первой строке находится число n - количество элементов в массиве. Во второй строке находятся n чисел - элементы массива. Все числа во входных данных не превышают 1000. Описание выходных данных:
Выведите элементы массива в обратном порядке.
Входные данные
3
1 2 3
Выходные данные
3 2 1
