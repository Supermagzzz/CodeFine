E. Преобразование (35 б.)
Вам дан массив из n целых чисел. Необходимо найти число, которое встречается в этом массиве наибольшее количество раз, а если таких чисел несколько, то найти минимальное из них, и после этого переместить все такие числа в конец массива. Порядок расположения остальных чисел должен остаться без изменения.
Например, массив 1, 2, 3, 2, 3, 1, 2 после преобразования должен превратиться в массив 1, 3, 3, 1, 2, 2, 2.
Описание входных данных:
В первой строке находится число 1 ⩽ n ⩽ 100000.
Во второй строке находится сам массив из n чисел. Каждое число не превышает миллиона по модулю.
Описание выходных данных:
Выведите полученный после преобразования массив.
Входные данные
7
1 2 3 2 3 1 2
Выходные данные
1 3 3 1 2 2 2
