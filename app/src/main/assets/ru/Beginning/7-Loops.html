Циклы

До сих пор каждая команда выполняла одно действие. А что если вам нужно выполнить какую-то операцию миллион раз? Неужели придется писать миллион строчек кода? К счастью нет, для этого есть циклы.

for(инициализация цикла; условие продолжения; операция, которая выполняется после каждой итерации)
{
Код который будет повторяться
 }
Чаще всего этот цикл используют в следующей форме.
for(int i = 0; i < 1000; i++)
{
Код который будет повторяться
 }
Блок иницализации выполняется перед запуском цикла. В нем мы создаем переменную-счетчик i (такое название для переменной-счетчика обычно используется, если название переменной не должно нести дополнительной информации).
Дальше проверяется условие i < 1000. i равно нулю, так что код внутри выполнится.
После выполнения кода выполнится операция i++ - операция увеличения переменной i на единицу.
Теперь условие i < 1000 проверяется снова. Так как оно истинно, то код выполнится еще раз и i опять увеличится на единицу. Так будет до тех пор, пока переменная i не станет равна 1000, и условие не окажется ложным.

Если подвести краткий итог, то это цикл который выполнит код тысячу раз. А вот цикл который выполнит код миллион раз.

for(int i = 0; i < 1000000; i++)
{
Код который будет повторяться
 }
Давайте напишем код, который выведет текст "Hello" 20 раз.
for(int i = 0; i < 20; i++)
{
cout << "Hello" << endl;
 }
Вы также можете использовать переменную-счетчик внутри цикла. Давайте выведем все натуральные четные числа до N включительно.
int N = 1000;
for(int i = 2; i <= N; i += 2)
{
cout << i << " ";
 }
Что делает этот код? Инициализация начинается с 2, так как это первое четное натуральное число. Следующее число будет на два больше, поэтому после итерации будет выпольняться код i += 2. Так как все числа должны быть меньше либо равны N, то условие продолжения i <= N.
Посмотрим на еще один пример применения циклов. Пусть у вас есть результаты выборов в какой-нибудь стране и вы хотите узнать кто победил. На вход программе подается число N - количество голосующих. В следующих N строках будет находится число 1 или 2, которое означает за какого кандидата был отдан голос.
Давайте найдем результаты выборов.
#include <iostream>

using namespace std;

int main()
{
cout.precision(2);
cout << fixed;
int N;
cin >> N;
int countFirst = 0, countSecond = 0;
for(int i = 0; i < N; i++)
{
int x;
cin >> x;
if (x == 1)
{
countFirst++;
}
else
{
countSecond++;
}
}
cout << "1: " << countFirst * 1.0 / N * 100 << "%" << endl;
cout << "2: " << countSecond * 1.0 / N * 100 << "%" << endl;
}
В этом коде мы считываем каждый голос и проверяем за кого он был отдан. Когда все голоса были посчитаны, мы находим их долю от общего числа голосов (countFirst * 1.0 / N). Не забывайте, что целые числа делятся с округлением вниз, поэтому чтобы разделить их без округления нужно превратить их в double. Если умножить целое число на вещественное, то результат будет вещественный. При умножении числа на единицу значение этого числа не меняется. Значит x * 1.0 превратит целый x в вещественное число.
Теперь умножим полученное значение на 100, чтобы перевести его в проценты. Так мы получим процент людей проголосовавших за каждого кандидата.
Запустите эту программу и введите следующие данные.
3
1
2
2
 Они означают, что у вас есть три голоса и первый человек проголосовал за первого кандидата, а остальные за второго. Программа выведет следующий результат.
1: 33.33%
2: 66.67%

Теперь давайте решим более сложную задачу. У вас есть число x. Давайте проверим простое ли оно. Простое число это то, которое делится только на себя и на единицу.
#include <iostream>

using namespace std;

int main()
{
int x;
cin >> x;
bool isPrime = true;
for(int i = 2; i < x; i++)
{
if (x % i == 0)
{
isPrime = false;
break;
}
}
if(isPrime)
{
cout << "Yes";
}
else
{
cout << "No";
}
}

Давайте разберем этот код. Сначала решим задачу с точки зрения математики. Число никогда не может делиться на большее число. Значит нам нужно перебрать все числа от двух до x - 1 и проверить делится ли x на это число.
Если мы найдем число, на которое делится x, значит x не простое. Как проверить делится ли одно число на другое? Если число делится, то остаток от деления будет равен нулю.
Приступим к коду. Переменная isPrime будет содержать информацию о простоте числа. Давайте сначала предположим что число простое, а потом попытаемя доказать обратное. Пройдемся циклом по числам от 2 до x не включительно и проверим условие x % i == 0. Если вдруг это условие истинно то мы говорим, что число не простое isPrime = false;.
А в следующей строчке вы можете увидеть новую команду break. Эта команда заканчивает выполнение текущего цикла. Так как мы уже нашли один делитель, то нам не имеет смысла проверять все остальные числа. С помощью этой команды можно ускорить среднее время работы вашего кода. Также существует команда continue, которая сразу начинает выполнение следующей итерации, сразу прерывая эту.
Когда цикл завершится в переменной isPrime будет находиться ответ на вопрос - "простое ли число x?".

Вложенные циклы

Давайте улучшим прошлое решение решив следующую задачу. Выведите все простые числа до N. Для этого можно создать цикл, который перебирает значения x и скопировать код прошлого решения туда, как блок кода.

#include <iostream>

using namespace std;

int main()
{
int N;
cin >> N;
for(int x = 2; x <= N; x++)
{
bool isPrime = true;
for(int i = 2; i < x; i++)
{
if (x % i == 0)
{
isPrime = false;
break;
}
}
if(isPrime)
{
cout << x << " ";
}
}
}
Как вы можете видеть из этого примера циклы можно вкладывать друг в друга. Только не используйте одинаковые имена для переменных-счетчиков, ваша программа может начать работать неккоректно.
Вложенные циклы могут показаться очень сложными для новичков, но стоит разобраться с ними. Это очень важный инструмент в программировании.
Давайте напишем программу, которая выведет на экран таблицу умножения. Будем выводить строки по очереди, так что давайте сначала научимся выводить строку с номером numRow на экран. В строке с номером x находятся числа 1 * x, 2 * x, 3 * x и т.д
for(int numCol = 1; numCol <= 10; numCol++)
{
cout << numRow * numCol << " ";
}
numRow - переменная с номером строки, а numCol - переменная с номером столбца.
Теперь мы можем вывести одну строку. Давайте поместим этот цикл в другой, который будет перебирать номер строки.
#include <iostream>

using namespace std;

int main()
{
for(int numRow = 1; numRow <= 10; numRow++)
{
for(int numCol = 1; numCol <= 10; numCol++)
{
cout << numRow * numCol << " ";
}
cout << endl;
}
}
Этот код выведет на экран таблицу умножения. Не забывайте, что как только вы вевели одну строку, то нужно сразу сделать перенос.

Другие типы циклов

Помимо цикла for существует еще два типа циклов. Первый из них это while. Этот цикл выглядит как if, но выполняет блок кода внутри не один раз, а пока условие истинно.

#include <iostream>

using namespace std;

int main()
{
int a = 0;
while(a < 3)
{
cout << a << endl;
a++;
}
}
Этот код выведет на экран числа 0, 1 и 2. Сейчас цикл while не сильно отличается от цикла for. Но его удобно использовать именно для проверки выполнения условий. Например вот код, который находит сумму цифр числа.
#include <iostream>

using namespace std;

int main()
{
int x, sum = 0;
cin >> x;
while(x > 0)
{
sum += x % 10;
x /= 10;
}
cout << sum;
}
Остаток от деления числа на 10 это последняя цифра числа, а деление числа на десять нацело это удаление последней цифры числа. Вот пример.
512 % 10 = 2
512 / 10 = 51
 Как только число стало равно нулю, значит мы удалили у него все цифры. Значит код выше находит сумму всех цифр числа.
Используя свойства цикла while можно легко получить бесконечный цикл - while(true). Главное не забудьте добавить условие выхода break внутри блока кода этого цикла. Зачем это нужно? Вот пример программы, которая проверяет пароль пользователя.
#include <iostream>
#include <string>

using namespace std;

int main()
{
string password = "1234";
while(true)
{
string input;
cin << input;
 if(input == password)
{
cout << "Welcome" << endl;
break;
}
else
{
cout << "Wrong password" << endl;
}
}
}
Этот код будет выполняться до тех пор, пока пользователь не введет верный пароль. Как только верный пароль будет введен цикл завершится командой break.
Еще один тип цикла это do{блок кода} while(условие);. Он почти никогда никем не используется, так что мы просто расскажем вам о нем без примеров.
Этот цикл сначала выполняет блок кода, а потом только проверяет условие внутри while. Если условие истинно, то код выполнится еще раз.
#include <iostream>

using namespace std;

int main()
{
int a = 0;
do
{
a++;
cout << a << endl;
}
while(a < 3);
}
Этот код выведет на экран числа 1, 2 и 3.
