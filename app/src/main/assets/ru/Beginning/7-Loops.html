Циклы

До сих пор каждая команда выполняла одно действие. Но, что делать, если необходимо выполнить какую-то операцию миллион раз? Писать миллион строчек кода? К счастью, нет. Для этой задачи существуют циклы.

for(инициализация цикла; условие продолжения; переход к новой итерации)
{
Код который будет повторяться
 }

Чаще всего циклы используются в следующей форме:

for(int i = 0; i < 1000; i++)
{
Код который будет повторяться
 }

1) Команда "иницализация цикла" выполняется перед запуском цикла. Она создает переменную-счетчик i равную нулю.
int i = 0;

2) Перед выполением кода внутри цикла, проверяется "условие продолжения". В нашем случае это условие, что переменная-счетчик меньше тысячи. Так как переменная равна нулю, то "условие продолжения" верно. Если условие было бы ложно, то выполнение цикла бы завершилось.
i < 1000;

3) Выполняется код внутри цикла.

4) Выполняется команда "переход к новой итерации". В примере выше, в этой команде мы прибавляем к переменной-счетчику единицу. После этого, выполнение кода вновь возвращается к шагу 2, с новым значением переменной i. Нетрудно понять, что цикл завершится, когда переменная-счетчик станет равна тысяче. Значит, код внутри цикла выполнится ровно тысячу раз.
i++

А вот цикл, который выполнит код миллион раз.

for(int i = 0; i < 1000000; i++)
{
Код который будет повторяться
 }

Давайте напишем программу, которая выведет текст "Hello" 20 раз, каждый раз с новой строки:

for(int i = 0; i < 20; i++)
{
cout << "Hello" << endl;
 }

Можно использовать переменную-счетчик внутри цикла. Давайте выведем все четные числа от двух до N включительно.

int N = 10;
for(int i = 2; i <= N; i += 2)
{
cout << i << " ";
 }

Программа выведет:
2 4 6 8 10
Что делает этот код? Первое значение переменной-счетика равно двум, так как это первое четное число. Следующее число будет на два больше (чтобы пропустить нечетное), поэтому после вывода числа на экран будет выполнена операция i += 2. Все числа должны быть меньше либо равны N, поэтому условие продолжения i <= N.

Рассмотрим еще один пример.
У вас есть голоса избирателей на выборах в какой-нибудь стране, и вы хотите узнать победителя. На вход программе подается число N - количество голосующих. В следующих N строках будет находится число 1 или 2, которое означает за какого кандидата был отдан голос (1 - за первого кандидата, 2 - за второго).
Давайте найдем победителя:

#include <iostream>

using namespace std;

int main()
{
cout.precision(2);
cout << fixed;
int N;
cin >> N;
int countFirst = 0, countSecond = 0;
for(int i = 0; i < N; i++)
{
int x;
cin >> x;
if (x == 1)
{
countFirst++;
}
else
{
countSecond++;
}
}
cout << "1: " << countFirst * 1.0 / N * 100 << "%" << endl;
cout << "2: " << countSecond * 1.0 / N * 100 << "%" << endl;
}
Алгоритм решения задачи следующий: мы должны считать очередной голос, узнать за кого он был отдан и прибавить единицу к переменной, хранящей количество голосов, отданных за этого кандидата. Такой алгоритм мы должны повторить для каждого голоса, то есть N раз. С этой задачей хорошо справится цикл.

Сначала мы считываем общее количество голосов.
int N;
cin >> N;

Создаем две переменные, которые будут хранить количество голосов, отданных за каждого кандидата. Изначально у кандидатов нет ни одного голоса, поэтому запишем в эти переменные нули.
int countFirst = 0, countSecond = 0;

Теперь нам нужно обработать каждый голос. Всего голосов N, значит, мы должны провести алгоритм обработки одного голоса N раз. Создадим цикл.
for(int i = 0; i < N; i++)

Как выглядит алгоритм проверки одного голоса? Считаем голос, проверим, за кого он был отдан, и прибавим единицу к значению переменной нужного кандидата.
int x;
cin >> x;
if (x == 1)
{
countFirst++;
}
else
{
countSecond++;
}

Когда цикл завершится, мы будем знать количество голосов у каждого кандидата. Давайте найдем процент от общего числа голосов.
countFirst / N * 100
Этот код не будет работать, так как все значения целочисленные и результат деления будет округляться. Чтобы исправить это, превратим первое число в вещественное. Для этого можно умножить его на 1.0. Результат умножения целого числа на вещественное будет вещественным, а само число при этом не изменится, так как умножается на единицу.
cout << "1: " << countFirst * 1.0 / N * 100 << "%" << endl;
cout << "2: " << countSecond * 1.0 / N * 100 << "%" << endl;

Запустите эту программу и введите следующие данные:
3
1
2
2
 У вас есть три голоса. Первый человек проголосовал за первого кандидата, а остальные за второго. Программа выведет следующий результат.
1: 33.33%
2: 66.67%

Решим еще одну сложную задачу, чтобы лучше понять работу с циклами. У вас есть число x. Давайте проверим, простое ли оно. Простое число - это то, которое делится только на себя и на единицу.
#include <iostream>

using namespace std;

int main()
{
int x;
cin >> x;
bool isPrime = true;
for(int i = 2; i < x; i++)
{
if (x % i == 0)
{
isPrime = false;
break;
}
}
if(isPrime)
{
cout << "Yes";
}
else
{
cout << "No";
}
}

Решим задачу с точки зрения математики. Число никогда не может делиться на большее число. Значит нам нужно перебрать все числа от двух до x - 1 и проверить, делится ли x хоть на одно из них.
Если мы найдем число, на которое делится x, значит x не простое. Как проверить, делится ли одно число на другое? Если A делится на B, то остаток от деления A на B равен нулю.

Приступим к коду. Переменная isPrime будет содержать информацию о простоте числа. Предположим, что число простое, а потом попытаемся доказать обратное. Пройдемся циклом по числам от 2 до x-1 и проверим делимость. Если делится, то мы говорим, что число не простое: isPrime = false;

В следующей строчке находится новая команда - break. Эта команда заканчивает выполнение цикла, несмотря на "условие продолжения". Так как мы уже нашли один делитель, то нам не имеет смысла проверять все остальные числа. Также существует команда continue, которая сразу начинает выполнение следующей итерации (повторения цикла), игнорируя код, находящийся ниже.

После завершения цикла в переменной isPrime будет находиться ответ на вопрос - "простое ли число x?".

Вложенные циклы

Давайте улучшим прошлое решение, решив следующую задачу. Выведите все простые числа до N.

Для решения такой задачи можно создать цикл, который перебирает значения x, и скопировать код прошлого решения внутрь (для каждого x проверить, простое ли оно).

#include <iostream>

using namespace std;

int main()
{
int N;
cin >> N;
for(int x = 2; x <= N; x++)
{
bool isPrime = true;
for(int i = 2; i < x; i++)
{
if (x % i == 0)
{
isPrime = false;
break;
}
}
if(isPrime)
{
cout << x << " ";
}
}
}
Из этого примера видно, что циклы можно вкладывать друг в друга. Не используйте одинаковые имена для вложенных переменных-счетчиков, ваша программа может начать работать некорректно.
Вложенные циклы могут показаться очень сложными для новичков, но стоит разобраться с ними. Это очень важный инструмент в программировании.

Давайте напишем программу, которая выведет на экран таблицу умножения.

Так как мы будем выводить строки по очереди, так что давайте сначала научимся выводить строку с номером numRow на экран. В строке с номером x находятся числа 1 * x, 2 * x, 3 * x и т.д
for(int numCol = 1; numCol <= 10; numCol++)
{
cout << numRow * numCol << " ";
}
numRow - переменная с номером строки, а numCol - переменная с номером столбца.
Теперь мы можем вывести одну строку. Давайте поместим этот цикл в другой, который будет перебирать номер строки.
#include <iostream>

using namespace std;

int main()
{
for(int numRow = 1; numRow <= 10; numRow++)
{
for(int numCol = 1; numCol <= 10; numCol++)
{
cout << numRow * numCol << " ";
}
cout << endl;
}
}
Этот код выведет на экран таблицу умножения. Не забывайте, что как только вы вывели одну строку на экран, то нужно сразу сделать перенос.

Другие типы циклов

Помимо цикла for существует еще два типа циклов. Первый из них это while. Этот цикл выглядит как if, но выполняет блок кода внутри не один раз, а пока условие истинно.

#include <iostream>

using namespace std;

int main()
{
int a = 0;
while(a < 3)
{
cout << a << endl;
a++;
}
}
Этот код выведет на экран числа 0, 1 и 2. Сейчас цикл while не сильно отличается от цикла for. Но его нужно использовать именно для проверки выполнения условий. Например, вот код, который находит сумму цифр числа:
#include <iostream>

using namespace std;

int main()
{
int x, sum = 0;
cin >> x;
while(x > 0)
{
sum += x % 10;
x /= 10;
}
cout << sum;
}
Остаток от деления числа на 10 - это последняя цифра числа, а деление числа на десять нацело - это удаление последней цифры числа. Пример:
512 % 10 = 2
512 / 10 = 51
 Как только число стало равно нулю, значит мы удалили у него все цифры. Код выше находит сумму всех цифр числа.
Используя свойства цикла while можно легко получить бесконечный цикл - while(true). Главное, не забудьте добавить условие выхода и break внутри.

Для чего могут быть необходимо бесконечные циклы? Ниже приведен пример программы, которая проверяет ввод пароля пользователя:
#include <iostream>
#include <string>

using namespace std;

int main()
{
string password = "1234";
while(true)
{
string input;
cin >> input;
 if(input == password)
{
cout << "Welcome" << endl;
break;
}
else
{
cout << "Wrong password" << endl;
}
}
}
Этот код будет выполняться до тех пор, пока пользователь не введет верный пароль. Как только правильный пароль будет введен, на экране появится надпись Welcome, и цикл завершится командой break. Если пользователь введет неверный пароль, то на экране появляется надпись Wrong password, и цикл начинается сначала.

Еще один тип цикла это do{блок кода} while(условие);. Он достаточно редко используется.
Этот цикл сначала выполняет блок кода, а потом проверяет условие внутри while. Если условие истинно, то код выполнится еще раз.
#include <iostream>

using namespace std;

int main()
{
int a = 0;
do
{
a++;
cout << a << endl;
}
while(a < 3);
}
Этот код выведет на экран числа 1, 2 и 3.
