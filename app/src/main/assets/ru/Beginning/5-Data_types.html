Типы данных

На прошлом уроке мы познакомились с переменными. Оказалось, что у каждой переменной есть свой тип. Мы изучили int - целочисленный тип данных. Но, что делать, если вам нужно хранить строку или вещественное число? Ответ на этот вопрос вы получите на данном уроке.

Вещественные типы данных
Чтобы хранить вещественные числа (поддерживающие цифры после запятой) существует тип float.

float a = 5.0;

Обратите внимание, что дробная часть числа записывается через точку.

С типом float можно производить обычные математические операции (сложение, умножение и д.р).
В отличии от int операция деления производится без округления результата (3.0 / 2.0 = 1.5), а операция остатка от деления невозможна.

Если вы не указываете дробную часть числа, то оно считается за целое. Например:
3 / 2 = 1
3.0 / 2.0 = 1.5
Но если хоть одно число вещественное, то результат вычисления тоже будет вещественным. Например:
3.0 / 2 = 1.5

Предположим, что мы хотим вывести вещественное число на экран. Сколько цифр после запятой выведется? Чтобы указать это количество необходимо использовать следующие команды:

cout.precision(3);
cout << fixed;

Команда cout.precision(x) указывает, что нужно выводить x цифр после запятой. Если вы укажете 0, то будут выводится целые числа.
Команда cout << fixed запрещает вывод чисел в научной форме. Например: 1.3e-10.

Посмотрите на код программы, которая читает два вещественных числа и выводит их сумму, разность, произведение и частное.

#include <iostream>

using namespace std;

int main()
{
cout.precision(3);
cout << fixed;
float a, b;
 cin >> a >> b;
 cout << a + b << endl;
 cout << a - b << endl;
 cout << a * b << endl;
 cout << a / b << endl;
 }

Главная проблема вещественных чисел состоит в том, что компьтер не умеет хранить их точно. Из-за этого в вычислениях появляется погрешность. Поэтому этот тип данных не стоит использовать для точных расчетов.

Проблема возникает из-за того, что на хранение каждого числа компьютер тратит определенное количество бит. Это сказывается также на работе с целыми типами данных. У каждого типа есть свои ограничения, насколько большие числа он может хранить. Например, тип int может хранить числа в пределах двух миллиардов.

Как же быть, если вам необходимы большие или более точные числа? Используйте другие типы данных.

На практике обычно используют int, а в случае необходимости больших чисел используют long long, жертвуя памятью, занимаемой программой. Для вещественных чисел обычно используют double.
Обычно long long (18 знаков) хватает, чтобы хранить любые числа, необходимые для работы вашей программы. Но если вам этого мало, то можно использовать алгоритмы длинной арифметики. Мы расскажем о них в третьем курсе нашего учебника.

Нельзя создавать переменные с разными типами данных в одной строке. Правильно делать так:
double a, b, c;
int d, e, f;

Существует также логический тип bool, который может хранить только значения 1 или 0 (правда или ложь). Это очень нужный тип, и скоро вы сами убедитесь в этом.

Строковые типы данных
Чтобы использовать тип данных, который может хранить строки, нужно подключить библиотеку string.
#include <string>
Теперь вы можете использовать тип string - строковый тип данных. Пример:
string a = "Hello world!";
Или
#include <iostream>
#include <string>

using namespace std;

int main()
{
string name;
 cin >> name;
 cout << "Hi, " << name << "!";
 }
Это программа считает ваше имя (например, Max) и выведет строку:
Hi, Max!

Также вы можете использовать тип char, хранящий всего один символ. Для использования этого типа не нужно подключать библиотеки.
char a = 'T';
Обратите внимание, что строки должны находиться в двойных кавычках, а символы в одинарных.
Давайте повторим весь пройденный материал написав программу-анкету.
#include <iostream>
#include <string>

using namespace std;

int main()
{
cout.precision(1);
 cout << fixed;
 string name, country;
 int age, favorite, countFriends;
cout << "What is your name?" << endl;
 cin >> name;
 cout << "Where are you live?" << endl;
 cin >> country;
 cout << "How old are you?" << endl;
 cin >> age;
 cout << "What is your favorite number?" << endl;
 cin >> favorite;
 cout << "How many friends do you invite to your party?" << endl;
 cin >> countFriends;
 cout << "Hi, " << name << " from " << country << "!" << endl;
 cout << "Sum of your age and favorite number is " << age + favorite << endl;
 cout << "If you have one cake at the party, everyone will get " << 1.0 / (countFriends + 1) * 100 << "% from it" << endl;
 }
Этот код считывает имя, страну, возраст, любимое число и количество друзей, и выводит некую информацию на основе этих данных. Внимательно разберите этот код самостоятельно.
