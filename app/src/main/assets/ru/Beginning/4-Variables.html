Переменные

На прошлом уроке мы научились выводить данные на консоль, чтобы их можно было увидеть пользователю. Но как можно получить данные от пользователя?
На в этом уроке мы попробуем написать свой простой консольный калькулятор, который умеет складывать, вычитать и умножать два числа.

Если вы получите данные от пользователя, то их нужно будет где-то хранить. Для этого у копьютера есть оперативная память.
Чтобы хранить данные в памяти, нужно создавать для них переменные (можете представить их себе, как коробки в которые, мы положим данные). Переменные создаются следующим образом:

#include <iostream>

using namespace std;

int main()
{
    тип переменной название переменной = данные;
}
У каждой переменной есть свой тип, который означает, какие именно данные мы будем хранить. Про то, какие типы существуют мы поговорим чуть позже, а сейчас поработаем с типом "целое число".
Чтобы хранить в памяти целое число, используется тип int.
Название переменной вы выбираете сами, оно необходимо, чтобы записывать или получать данные из нее (положить или достать данные из коробки).
При запуске программы, каждое название переменной в коде заменится на ее значение. Чтобы было понятнее, рассмотрим пример:
#include <iostream>

using namespace std;

int main()
{
    int a = 5;
    cout << a;
}
Этот код выведет на экран число 5, так как подставит в команду cout число вместо a.
Учтите, что a - это не текст, а переменная. Поэтому ее нельзя писать в кавычках.

Есть несколько правил, по которым подбираются имена для переменных:
1) Переменные могут состоять из английских букв, цифр и символа подчеркивания.
2) Цифра не может быть первым символом в имени переменной.
3) Нельзя давать переменным имена, которые совпадают с другими названиями в C++.
Например, запрещено давать такие имена, как int, cout, endl и т.д.

В одной программе можно создавать несколько переменных.

#include <iostream>

using namespace std;

int main()
{
    int a = 5;
    int b = 7;
    cout << a << " " << b;
}
Этот код выведет на экран первую переменную, пробел, затем вторую переменную. После завершения программы на экране будет:
5 7
Вернемся к задаче, поставленной в начале урока. Как получить значение переменной от пользователя? Для этого воспользуемся командой cin - вводом с консоли.
Ее нужно использовать также как и cout, но только стрелочки должны показывать в другую сторону.
#include <iostream>

using namespace std;

int main()
{
    int a = 0;
    cin >> a;
    cout << a;
}
После запуска программа ничего не выведет и будет ждать вашего ввода.
Введите какое-нибудь число и нажмите Enter. Программа прочитает его и запишет в переменную и выведет ее на экран.

Для чего записывать в переменную ноль при создании, если прямо сейчас пользователь запишет в нее другое значение? Действительно, можно создавать переменные не записывая в них значение сразу. Такой код тоже работает:
#include <iostream>

using namespace std;

int main()
{
    int a;
    cin >> a;
    cout << a;
}
Можно объявлять несколько переменных одного типа в одной строке через запятую. Учтите, что тип переменной при этом пишется только один раз в начале строки. При вводе данных для разных переменных в консоль нужно разделять их пробелом.

Запустите следующую программу и введите два числа через пробел. Программа напечатает эти два числа и число 25.
#include <iostream>

using namespace std;

int main()
{
    int a, b, c = 25;
    cin >> a >> b;
    cout << a << " " << b << " " << c;
}
К числовым переменным можно применять математические операции (сложение, умножение и др.) . В переменную можно записывать не только число, но и математическое выражение.
#include <iostream>

using namespace std;

int main()
{
    int a = 5 + 5 * 7;
    int b = a + a, c;
    cin >> c;
    b = b + c;
    cout << a << " " << b << " " << c;
}
Здесь стоить учесть еще одно правило. Значения переменных всегда можно изменять (положить новое значение в коробку). Тип переменной мы указываем только при ее создании (т.е. при первом объявлении). Если вы в дальнейшем записываете в переменную новое значение, то тип писать нельзя.
Давайте разберемся, что делает программа выше.
int a = 5 + 5 * 7;
1) Объявили переменную a равную 40.
int b = a + a;
2) Объявили переменную b равную 80 (40 + 40).
int c;
cin >> c;
3) Считали переменную c с консоли. (Предположим, что мы ввели число 20).
b = b + c;
4) Прибавили к переменной b число 20, теперь оно равно 100 (b = 80 + 20).
cout << a << " " << b << " " << c;
5) Вывели все три числа на экран. Результат:
40 100 20

Мы достаточно быстро разобрали основы работы с переменными, но это очень важная тема. Остановитесь и убедитесь, что вы понимаете, как работает программа выше и можете написать ее сами. Если возникают трудности, советуем перечитать статью и внимательнее изучить примеры с кодом.

Математические операции

Давайте рассмотрим основные арифмитические операции с целыми числами.
с = a + b; Операция сложения двух чисел.
с = a - b; Операция вычитания двух чисел.
с = a * b; Операция умножения двух чисел.
С делением все не так просто. Так как результат деления может быть не целым, то мы не сможем записать его в целый тип данных. В следующем уроке мы рассмотрим решение этой проблемы и узнаем про другие типы данных.
У типа int есть следующие операции.
с = a / b; Целая часть от деления a на b (Пример: 10 / 3 = 3, 15 / 3 = 5).
с = a % b; Остаток от деления a на b (Пример: 10 % 3 = 1, 15 % 3 = 0).
Если вы хотите прибавить к переменной число или применить к ней любую другую операцию, то необязательно писать так:
a = a + b;
Вместо этого можно воспользоваться операцией прибавления:
a += b; (Если a было равно 5, а b было равно 3, то a станет равно 8).
Также можно делать с любой операцией. Примеры:
a -= 2;
a *= b;
a /= b - c + 35 * 60;
Если вы хотите прибавть или отнять единицу от числа, то можно воспользоваться следующей формой.
a++; (Прибавляет 1 к a)
a--; (Отнимает 1 от a)

Обладая всеми этими знаниями, нам не составит труда написать свой калькулятор. Давайте напишем его и одновременно научимся решать наши задачи, где нужно считывать данные.
Задача:

На вход подаются два целых числа. Выведите их сумму, разность и произведение, каждое число должно быть с новой строки.

Входные данные
5 3
Выходные данные
8
2
15
Сначала идет условие задачи, потом пример тестов (входных и выходных данных), чтобы показать, как должна работать ваша программа. По условию ваша программа должна считать два числа и вывести три числа, каждое в новой строке. Давайте напишем такую программу.
#include <iostream>

using namespace std;

int main()
{
    int a, b;
    cin >> a >> b;
    cout << a + b << endl;
    cout << a - b << endl;
    cout << a * b << endl;
}
Именно так нужно оформлять решения задач из учебника. После решения задачи, перейдите по ссылке, расположеной под ее условием и отправьте код на проверку. Подробнее можете узнать в уроке "Установка ПО".
Напоследок, давайте добавим поясняющий текст к нашей программе, чтобы пользователю было удобнее использовать ваш калькулятор (так вы можете делать в своих личных программах, но при решении задач из учебника придерживайтесь формата данных из условия. Не выводите ничего лишнего. То, что выведет ваша программа на ввод из Входных данных, должно полностью совпадать с тем, что находится в Выходных данных).
#include <iostream>

using namespace std;

int main()
{
int a, b;
 cout << "Enter two fractional numbers" << endl;
 cin >> a >> b;
 cout << "The sum is " << a + b << endl;
 cout << "The difference is " << a - b << endl;
 cout << "The product is " << a * b << endl;
 }
Чтобы закрепить пройденный материал, решите задачи на использование целочисленных переменных.
