Первая программа
Код из Codeblocks:
#include <iostream>

using namespace std;

int main()
{
    cout << "Hello world!" << endl;
    return 0;
}

При первом открытии Codeblocks вы увидели эту программу. Давайте разберемся, как она работает.

Сейчас вы можете считать, что весь код, отвечающий за работу программы, находится внутри фигурных скобок после строки int main()
(Вскоре вы узнаете, что это не совсем так).

int main()
{
    Ваш код будет находиться тут.
}

В данном примере за работу программы отвечают следующие две строки:

cout << "Hello world!" << endl;
return 0;

Первая из них выводит надпись "Hello world!" на экран. Давайте разберем ее подробнее.

Команда cout - это команда вывода данных на экран. Чтобы использовать ее, выполните следующие действия.
1) Наберите cout в начале строки.
2) Затем наберите << и информацию, которую вы хотите вывести (в кавычках). Можете представлять что вы стрелочками указываете данным на cout.
3) Вы можете повторить шаг 2 несколько раз. Например, в примере выше, мы сначала выводим на экран текст Hello world!, а потом endl. endl - это перенос строки. Все данные, которые вы выведете после него, появятся с новой строки. Значит, мы выводим на экран "Hello world" и переносим строку.
4) Поставьте точку с запятой в конце строки. Она должна находиться после каждой команды.

Давайте попробуем изменить программу. Напишите свое имя вместо "Hello world!". Учтите, что все текстовые данные в языке C++ должны находиться внутри двойных кавычек. Также советуем пользоваться только английским алфавитом, так как другие языки, возможно, не будут корректно отображаться (Позже мы покажем, как это исправить).
Пример:

#include <iostream>

using namespace std;

int main()
{
    cout << "Max" << endl;
    return 0;
}

Запустив код, вы увидите свое имя на экране.

Данные выводятся на экран в том порядке, в котором вы передали их с помощью символов <<.
Например, прошлую программу можно было переписать так:

#include <iostream>

using namespace std;

int main()
{
    cout << "M" << "a" << "x" << endl;
    return 0;
}

Этот код также выводит строку Max на экран. Результат работы программы не изменится.
Теперь попробуем использовать endl (символ переноса строки). Все, что вы будете выводить после него, отобразится на следующей строке. Чтобы понять лучше, запустите следующий код.

#include <iostream>

using namespace std;

int main()
{
    cout << "Line 1" << endl;
    cout << "Line 2" << endl << "Line 3";
    return 0;
}

Результат работы будет таким:

Line 1
Line 2
Line 3

Как можно увидеть из этого примера, необязательно выводить всю информацию в одной команде. Можно использовать cout сколько угодно раз в любом месте вашего кода. Данные будут выводиться на экран в том порядке, в котором вы передали их с помощью символов <<. Если в вашей программе используется несколько команд, то они будут выполняться по очереди, сверху вниз.
Например, можно переписать пример с именем ещё и таким способом:

#include <iostream>

using namespace std;

int main()
{
    cout << "M";
    cout << "a";
    cout << "x";
    return 0;
}

Давайте вернемся к первому примеру (код из Codeblocks). Что означает команда return 0;? Это завершение вашей программы. Весь код после этой строки будет игнорироваться.

#include <iostream>

using namespace std;

int main()
{
    cout << "Line 1";
    return 0;
    cout << "Line 2";
}

На экран будет выведено только Line 1.

Если же вы хотите, чтобы весь код вашей программы выполнился полностью, то необязательно писать return 0; . Этот код также является правильным.

#include <iostream>

using namespace std;

int main()
{
    cout << "Hello world!" << endl;
}

Мы разобрались, как работает программа из Codeblocks, которая выводила Hello world! на экран. Теперь разберемся, что делают строчки, находящиеся раньше int main().

#include <iostream>

using namespace std;

Изначально в языке C++ очень мало встроенных функций. Это сделано для того, чтобы ваши программы быстро компилировались и занимали мало памяти. Если вам нужна какая-то дополнительная функциональность, то ее можно подключить с помощью библиотек (специальных файлов, которые расширяют возможности языка).

Библиотека iostream добавляет возможность выводить данные на экран.
Если не подключить эту библиотеку, то компилятор не будет знать, что такое cout и endl, и вы не сможете использовать их в своей программе.
Для подключения библиотеки, используется строка:

#include <Название подключаемой библиотеки>

Теперь посмотрим на строку using namespace std;
Все функции, которые вы дополнительно подключаете с помощью библиотек, имеют приставку std:: . Строка using namespace std; убирает эту приставку. Если не писать эту строку, то код будет выглядеть так:

#include <iostream>

int main()
{
    std::cout << "Hello world!" << std::endl;
    return 0;
}

Вы можете сами решить, оставлять приставку std:: или нет, но для обучения, будет удобнее убрать ее. Мы советуем делать так при решении наших задач.
