D. Треугольник

Треугольник может существовать только если каждая из его сторон меньше суммы двух оставшихся.

Решение задачи:

#include <iostream>

using namespace std;

int main()
{
int a, b, c;
cin >> a >> b >> c;
if(a < b + c && b < a + c && c < a + b)
{
cout << "Yes";
}
else
{
cout << "No";
}
}
