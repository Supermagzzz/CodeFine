D. Переключение гравитации

Если кубик находится в строке, то он останется в ней же после переключения гравитации. Изменится только столбец. Значит числа в массиве измениться не могут. Можно заметить, что решением задачи является отсортированный массив.

Решение задачи:

#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;
int main()
{
int n;
cin >> n;
vector<int> v(n);
for(int &e : v)
{
cin >> e;
}
sort(v.begin(), v.end());
for(int e : v)
{
cout << e;
}
}
