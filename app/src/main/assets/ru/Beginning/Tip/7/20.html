D. Загадка

Переберем все возможные ответы и найдем подходящий. Для этого переберем X, а внутри этого цикла для каждого X, переберем все возможные Y. Когда правильный ответ найден, можно завершать программу. Так как второе число должно быть больше либо равно первому, то Y стоит перебирать не с единицы, а с X;

Решение задачи:

#include <iostream>

using namespace std;

int main()
{
int S, P;
cin >> S >> P;
for(int X = 1; X <= 100; X++)
{
for(int Y = X; Y <= 100; Y++)
{
if(X + Y == S && X * Y == P)
{
cout << X << " " << Y;
return 0;
}
}
}
}
