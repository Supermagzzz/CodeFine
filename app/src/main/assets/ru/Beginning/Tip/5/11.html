B. Большое умножение

Так как у нас есть два числа до миллиарда, значит их произведение будет до 10^9 * 10^9 = 10^18. Такое большое число не поместится в тип int, поэтому для решения задачи нужно использовать тип long long.

Решение задачи:

#include <iostream>

using namespace std;

int main()
{
long long a, b;
cin >> a >> b;
cout << a * b << endl;
}
