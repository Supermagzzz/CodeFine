Массивы

Представьте, что вы хотите хранить в программе большое количество данных. Например, вспомните задачу про голосование из прошлой статьи. Что если вам нужно будет сохранить в памяти все голоса, чтобы обработать их в дальнейшем? Для этого существуют массивы. Массивы - это много переменных объединенных одним именем.

Решим следующую задачу. У вас есть голоса с выборов, в которых учавстовали 40 кандидатов, и вы хотите узнать, кто победил. Конечно, вы можете решить эту задачу, используя обычные переменные (всего 40 штук), но код будет огромным и сложным. С помощью массивов написать такую программу проще.

Для создания массива используется следующая конструкция:

#include <iostream>

using namespace std;

int main()
{
int candidates[40];
}

Создание массива не сильно отличается от создания переменной. После имени массива в скобках нужно указать количество элементов(переменных) в нем. Как мы знаем, переменные нужно инициализировать, если вы не собираетесь сейчас их считывать. С массивом все точно также, но чтобы инициализировать его, нужно использовать следующую конструкцию:

#include <iostream>

using namespace std;

int main()
{
int a[3] = {0, 0, 15};
}

Этот код создаст массив из трех чисел 0, 0 и 15. Все числа должны быть заключены в фигурные скобки.

Как получить переменную из массива? У каждой переменной есть свой номер. У первой переменной номер 0, у второй переменной номер 1 и т.д. Чтобы получить переменную, нужно просто написать имя массива и номер этой переменной в квадратных скобках. Например, вторая переменная в массиве a это a[1].

Номер переменной называется индексом.

#include <iostream>

using namespace std;

int main()
{
int a[3] = {0, 0, 15};
cout << a[2];
}
Этот код выведет 15.

Давайте вернемся к первой задаче. Нам нужно инициализировать массив, присвоив каждой переменной ноль. Но в нем целых 40 значений. Чтобы не делать много повторяющихся действий, можно заменить их циклами.

#include <iostream>

using namespace std;

int main()
{
int candidates[40];
for(int i = 0; i < 40; i++)
{
candidates[i] = 0;
}
}
Мы прошлись циклом по каждой переменной и присвоили им значение ноль. Цикл for и массивы идеально сочетаются друг с другом, ведь переменная-счетчик может легко пробежать по всем индексам массива.

Пусть нам на вход дают число N - количество проголосовавших. Дальше будут записаны N чисел. Каждое число - это номер кандидата, за которого проголосовал человек. Например, 1 - первый кандидат, а 40 - сороковой кандидат. Давайте посчитаем количество голосов за каждого кандидата.

#include <iostream>

using namespace std;

int main()
{
int candidates[40];
for(int i = 0; i < 40; i++)
{
candidates[i] = 0;
}
int N;
cin >> N;
for(int i = 0; i < N; i++)
{
int x;
cin >> x;
 candidates[x - 1]++;
}
}

Так как кандидаты пронумерованы с единицы, а индексы массива с нуля, то нужно отнять единицу от номера кандидата.

Теперь мы знаем количество голосов за каждого кандидата. Давайте выведем результаты голосования.

for(int i = 0; i < 40; i++)
{
cout << i + 1 << ": " << candidates[i] * 1.0 / N * 100 << "% " << endl;
}

При выводе номера кандидата мы возвращаем единицу к индексу. Остановитесь на этом моменте и убедитесь, что вы понимаете код выше. Если возникают трудности, то посмотрите на решение подобной задачи с двумя кандидатами в прошлой статье, перечитайте эту статью и вернитесь к этому примеру. Вы должны понимать, как массив заменил переменные.

Если затруднений у вас не возникло, то давайте доработаем программу. Пусть на вход теперь подается число 1 <= M <= 100 - количество кандидатов. К сожалению, нельзя задавать размер массива с помощью переменной. Скоро мы изучим динамические массивы, в которых можно изменять размер, но на текущий момент, мы этого не умеем. Поэтому размер массива стоит задать достаточно большим, чтобы он вмещал всю нужную информацию. Лишние переменные мы просто не будем использовать. Но нельзя создавать слишком большие массивы, ведь они хранятся в вашей оперативной памяти, которая ограничена. Обычно массивы больше миллиона элементов не требуются, а это число легко помещается в память.

Из условий задачи мы видим, что кандидатов никогда не будет больше, чем 100. Значит размер массива можно задать равным сотне, таким образом туда поместятся все кандидаты. Перепишем код следующим образом:

#include <iostream>

using namespace std;

int main()
{
cout.precision(2);
cout << fixed;
int M;
cin >> M;
int candidates[100];
for(int i = 0; i < M; i++)
{
candidates[i] = 0;
}
int N;
cin >> N;
for(int i = 0; i < N; i++)
{
int x;
cin >> x;
 candidates[x - 1]++;
}
for(int i = 0; i < M; i++)
{
cout << i + 1 << ": " << candidates[i] * 1.0 / N * 100 << "% " << endl;
}
}

Изменилось немного, теперь в циклах мы идем не до 40, а до значения M. Давайте еще немного доработаем эту программу. Пусть теперь во входных данных, сразу после числа M, идут M имен кандидатов, каждое с новой строки.

В финальных списах мы теперь будем выводить не номер, а имя. Для этого создадим массив строк, в которой для каждого номера запишем имя.

#include <iostream>
#include <string>

using namespace std;

int main()
{
cout.precision(2);
cout << fixed;
int M;
cin >> M;
string names[100], a;
int candidates[100];
for(int i = 0; i < M; i++)
{
candidates[i] = 0;
cin >> names[i];
}
int N;
cin >> N;
for(int i = 0; i < N; i++)
{
int x;
cin >> x;
 candidates[x - 1]++;
}
for(int i = 0; i < M; i++)
{
cout << names[i] << ": " << candidates[i] * 1.0 / N * 100 << "% " << endl;
}
}

Строки тоже массивы
На самом деле string представляет из себя массив типа char. Поэтому можно, например, получить пятый символ в строке s[4]. Символы в строке можно изменять также, как и в массиве. Например, этот код выведет HelXo:
string a = "Hello";
a[3] = 'X';
cout << a;
Интересный факт, который поможет вам в будущем. Пусть у вас есть символ x. Из-за того, что все символы кодируются в системе ASCII, и все цифры и буквы находятся рядом друг с другом в порядке возрастания, то:
Если x - цифра, то x - '0' это численное значение этой цифры.
Если x - заглавная буква, то x - 'A' + 1 это ее номер в алфавите.
Если x - прописная буква, то x - 'a' + 1 это ее номер в алфавите.
Код находящий сумму цифр числа может выглядеть так:
string a = "15782";
int ans = 0;
for (int i = 0; i < a.size(); i++)
{
ans += a[i] - '0';
}
cout << ans;
Из этого кода вы также можете заметить новую функцию a.size(). Эта функция возвращает размер строки. К сожалению, у массива похожей функции нет.
Многомерные массивы
Вы можете легко создать двухмерный массив (табличка), просто добавив еще одни квадратные скобки к массиву.

int a[2][2] = {{0, 1}, {3, -1}};

Так как двухмерный массив - это по сути массив массивов чисел, то инициализация будет выглядеть именно так. Получать элементы можно также, как и в одномерном массиве, только используя вторые квадратные скобки. В a[1][0] лежит тройка. Вы также можете использовать и больше измерений, если захотите.
Инициализировать многомерные массивы можно также с помощью цикла.

int a[15][30][20];
for (int i = 0; i < 15; i++)
{
for (int j = 0; j < 30; j++)
 {
 for (int k = 0; k < 20; k++)
 {
 a[i][j][k] = 0;
}
 }
 }

Наверное, многомерные массивы не окажутся особо полезными для вас сейчас. Но когда вы начнете изучать алгоритмы, о которых мы рассказываем в нашем втором курсе, вы поймете насколько это важный элемент программирования.
