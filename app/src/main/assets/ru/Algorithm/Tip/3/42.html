B. Новый год: встреча друзей

Если отсортировать точки, то встреча должна произойти в центральной. Пусть точки называются A, B, C и B - центральная. N - расстояние между самыми дальними домами - |A - C|
Если они встрется в точке B, то суммарное расстояние равно |A - B| + |C - B| = N. А если в какой-то другой точке D, то |A - D| + |C - D| + |B - D| = N + |B - D|, что явно больше.

Решение задачи:

#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

int main()
{
vector<int> v(3);
for(int &e : v)
{
cin >> e;
}
sort(v.begin(), v.end());
cout << v[1];
}
