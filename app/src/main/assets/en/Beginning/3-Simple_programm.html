First program
Code from Codeblocks:
#include <iostream>

using namespace std;

int main()
{
    cout << "Hello world!" << endl;
    return 0;
}

When you first open Codeblocks, you will see this. Let’s analyze how it works.

You can think that all code that makes program works is in figured brackets after int main() string.
(You will understand that it’s not absolutely true soon)

int main()
{
    Your code will be here.
}

In this example, the following two lines are responsible for the working process of the program:

cout << "Hello world!" << endl;
return 0;

The first one prints “Hello world!” on the screen. Let’s investigate it.

Command cout - its a command to print data on the screen. To use it, follow next steps.
1) Write cout at the beginning of the line.
2) Next, write << and information that you want to display (in quotes). You can think that you use these arrows (<<) for point to cout for data.
3) You can repeat step 2 several times. For example, as in the case earlier. Firstly, we outputing Hello world! on-screen and than endl. endl is a line-breaker. All data which you enter after will be on the next line. So, we printing “Hello world” on screen and wrapping the line.
4) Write semicolon in the and of the line. It should be after every command.

Let's try to change the program. Write your name instead of "Hello world!" Keep in mind that all text data in C++ should be in quotes. Also, we recommend using only the English alphabet because other languages can be displayed incorrectly (we will show how to fix it later).
Example:

#include <iostream>

using namespace std;

int main()
{
    cout << "Max" << endl;
    return 0;
}

After launching this code you will see your name on the screen.

Data is printed on the screen with the same order as you wrote it after symbol <<.
For example, you can rewrite last program this way:

#include <iostream>

using namespace std;

int main()
{
    cout << "M" << "a" << "x" << endl;
    return 0;
}

This code is also prints on screen string Max. The result of program will be the same.
Now, let’s try to use endl (symbol for breaking lines). Launch this code for better understanding.

#include <iostream>

using namespace std;

int main()
{
    cout << "Line 1" << endl;
    cout << "Line 2" << endl << "Line 3";
    return 0;
}

There is a result:

Line 1
Line 2
Line 3

As we can see from this example, it is unnecessary to put all the information for output to one command. It is possible to use cout as much and as often as wanted. Data is displayed on the screen with the same order as you writing it after symbol <<. If your program uses some commands, they will be running in turn, from top to down.
For example, we can rewrite the case with a name this way:

#include <iostream>

using namespace std;

int main()
{
    cout << "M";
    cout << "a";
    cout << "x";
    return 0;
}

Let’s return to the first example (code from Codeclocks). What does command return means? It’s end in your program. All code after this string will be ignored.

#include <iostream>

using namespace std;

int main()
{
    cout << "Line 1";
    return 0;
    cout << "Line 2";
}

There will be Line 1 on the screen.

If you would like all code in your program to be fully completed, it’s not necessary to write return 0; This code is also correct.

#include <iostream>

using namespace std;

int main()
{
    cout << "Hello world!" << endl;
}

We figured out how program from Codeblocks which prints out Hello world! works. Let’s make clear what strings before int main() do.

#include <iostream>

using namespace std;

Initially, C++ has a little number of built-in functions. It was created to makes your programs fast-build and reduce memory usage. If you need extra functionality, you can add it using libraries (special files which empower language abilities).

Library iostream adding the capability to print data on a screen.
If you don't add this library, the compilator won’t know what cout and endl are, and you can not use them in your program.
For adding library use this string:

#include <Name of library>

Look at using namespace std; string.
All functions that you adding additionally using libraries have prefix std:: String using namespace std; deleting this prefix. If you don’t write this string code will look like this:

#include <iostream>

int main()
{
    std::cout << "Hello world!" << std::endl;
    return 0;
}

You can choose to save prefix std:: or not, but for studying it will be easier to delete it.We recommend doing this when you solving our tasks.
