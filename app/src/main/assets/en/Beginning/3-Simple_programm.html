Первая программа
Код из Codeblocks:
#include <iostream>

using namespace std;

int main()
{
    cout << "Hello world!" << endl;
    return 0;
}

Разберем программу, которую вы увидели при открытии Codeblocks.

Сейчас вы можете думать, что весь код программы будет находиться внутри фигурных скобок после строки int main()
(Скоро вы узнаете, что это не совсем так).

int main()
{
    Ваш код будет находиться тут.
}

Значит, за работу программы отвечают следующие две строчки.

cout << "Hello world!" << endl;
return 0;

Первая из них выводит надпись "Hello world!" на экран.
Команда cout (Console Out) - это команда вывода в консоль.
Стрелочками << вы отправляете "Hello world!" на вывод.
endl - это символ переноса строки. Отправив его в cout, дальнейший вывод в консоль продолжается с новой строки.

Каждая строка программы должна заканчиваться символом ; , так компилятор понимает где она заканчивается.

Давайте попрактикуемся. Напишите свое имя вместо "Hello world!". Учтите, что все текстовые данные в языке C++ должны находиться внутри двойных кавычек. Также советуем пока пользоваться только английским алфавитом, так как другие языки, возможно, не будут корректно отображаться (Позже мы покажем, как это исправить).
Пример:

#include <iostream>

using namespace std;

int main()
{
    cout << "Max" << endl;
    return 0;
}

Запустив код вы должны увидеть свое имя на экране.

Данные выводятся на экран в том порядке, в котором вы передали их с помощью символа <<.
Например, прошлую программу можно было написать так:

#include <iostream>

using namespace std;

int main()
{
    cout << "M" << "a" << "x" << endl;
    return 0;
}

Этот код выводит строку "Max" на экран по одному символу. Но результат работы программы не изменится.
Теперь попробуем использовать endl (символ переноса строки). Все, что вы будете выводить после него, будет отображаться на следующей строке. Чтобы понять лучше, запустите следующий код.

#include <iostream>

using namespace std;

int main()
{
    cout << "Line 1" << endl;
    cout << "Line 2" << endl << "Line 3";
    return 0;
}

Как можно увидеть из этого примера, необязательно выводить всю информацию на экран в одной команде. Можно использовать cout в любом месте, сколько угодно раз. A текст будет выводиться в том порядке, в котором вы передали его с помощью стрелочек.
Например, можно вывести свое имя ещё и таким способом.

#include <iostream>

using namespace std;

int main()
{
    cout << "M";
    cout << "a";
    cout << "x";
    return 0;
}

А что значит команда return 0; ? Это конец вашей программы. Весь код после этой строки будет игнорироваться.

#include <iostream>

using namespace std;

int main()
{
    cout << "Line 1";
    return 0;
    cout << "Line 2";
}

На экране будет выведено только "Line 1 ".
Но если вы хотите, чтобы весь код вашей программы выполнился, то необязательно писать return 0; . Этот код также является правильным.

#include <iostream>

using namespace std;

int main()
{
    cout << "Hello world!" << endl;
}

Мы разобрались, как работает программа из Codeblocks, которая выводила "Hello world! " на экран. Теперь разберем, что делают строчки, которые находятся раньше int main().

#include <iostream>

using namespace std;

Изначально в языке C++ очень мало возможностей. Это сделано для того, чтобы ваши программы были быстрыми и занимали мало места. Если вам нужна какая-то дополнительная функциональность, то ее можно подключить с помощью библиотек.
Библиотека iostream нужна для возможности использования ввода и вывода данных в консоль.
iostream - Input Output STREAM (Поток ввода и вывода).
Без подключения этой библиотеки C++ не будет знать, что такое cout и endl.
Для подключения библиотеки, используется команда:

#include <Название вашей библиотеки>

using namespace std; - это пространство имен. Позже мы расскажем, что это такое, но сейчас можно понимать эту команду так:
Все функции, которые вы дополнительно подключаете с помощью библиотек, имеют приставку std:: , а команда using namespace std; убирает эту приставку. Вам никто не запрещает писать и так:

#include <iostream>

int main()
{
    std::cout << "Hello world!" << std::endl;
    return 0;
}

У этих двух подходов есть свои плюсы и минусы на практике. Но, для обучения, убрать приставку std:: будет удобнее. Мы советуем делать так при решении наших задач.
