C. Расстояние между городами (8 б.)
Вам даны координаты двух городов. Найдите расстояние между ними. Расстоянием считается длина отрезка, соединяющего эти два города.
Описание входных данных:
В первой строке находятся координаты первого города −100 ⩽ x1, y1 ⩽ 100 - целые числа. Во второй строке находятся координаты второго города −100 ⩽ x2, y2 ⩽ 100 - целые числа.
Описание выходных данных:
Найдите расстояние между городами.
Входные данные
0 0
1 1
Выходные данные
1.414
Входные данные
0 0
0 5
Выходные данные
5.000
