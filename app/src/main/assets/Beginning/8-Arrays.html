Массивы

Представьте, что вы хотите хранить в программе огромный блок данных. Например вспомните задачу про голосование из прошлой статьи. Что если вам нужно будет сохранить в памяти все голоса, чтобы обработать их в дальнейшем? Для этого существуют массивы. Массивы - это блоки переменных объединенные одним именем.

Например давайте решим следующую задачу. На выборах представлено 40 кандидатов и вы хотите узнать кто победил. Конечно вы можете решить эту задачу используя обычные переменные, но код будет огронмый и сложный. Используя массивы все можно сделать намного проще. Чтобы создать массив используется следующая конструкция.

#include <iostream>

using namespace std;

int main()
{
int candidates[40];
}
Создание массива не сильно отличается от создания переменной. Нужно лишь после имени массива в скобках указать количество элементов в нем. Как мы знаем переменные нужно инициализировать, если вы не собираетесь сейчас их считывать. С массивом все точно также, но чтобы инициализировать его нужно использовать следующую конструкцию.
#include <iostream>

using namespace std;

int main()
{
int a[3] = {0, 0, 15};
}
Такой код создаст массив из трех чисел 0, 0 и 15. Все числа должны быть заключены в фигурные скобки.

Как получить переменную из массива? У каждой переменной есть свой номер. У первой переменной номер 0, у второй переменной номер 1 и т.д. Например вторая переменная в массиве a это a[1]. Чтобы получить переменную, нужно просто написать имя массива и номер этой переменной в квадратных скобках. Номер переменной называется индексом.

#include <iostream>

using namespace std;

int main()
{
int a[3] = {0, 0, 15};
cout << a[2];
}
Этот код выведет 15.

Давайте вернемся к нашей задаче. Нам нужно инициализировать массив присвоив каждой переменной ноль, но в нем целых 40 значений. Чтобы не делать руками много повторяющихся действий можно всегда заменить их циклами.

#include <iostream>

using namespace std;

int main()
{
int candidates[40];
for(int i = 0; i < 40; i++)
{
candidates[i] = 0;
}
}
Мы прошлись циклом по каждой переменной и присвоили ей значение ноль. Цикл for и массивы идеально сочетаются друг с другом, ведь переменная-счетчик может легко пробежать по всем индексам массива.

Теперь пусть нам на вход дают число N - число проголосовавших. Дальше будут записаны N чисел. Каждое число это номер кандидата за которого проголосовал человек. Например 1 - первый кандидат, а 40 - сороковой кандидат. Давайте посчитаем количество голосов за каждого кандидата.

#include <iostream>

using namespace std;

int main()
{
int candidates[40];
for(int i = 0; i < 40; i++)
{
candidates[i] = 0;
}
int N;
cin >> N;
for(int i = 0; i < N; i++)
{
int x;
cin >> x;
 candidates[x - 1]++;
}
}
Так как кандидаты пронумерованы с единицы, а индексы массива с нуля, то нужно не забыть отнять единицу от номера кандидата. Теперь мы знаем количество голосов за каждого кандидата. Давайте выведем результаты голосования.
#include <iostream>

using namespace std;

int main()
{
cout.precision(2);
cout << fixed;
int candidates[40];
for(int i = 0; i < 40; i++)
{
candidates[i] = 0;
}
int N;
cin >> N;
for(int i = 0; i < N; i++)
{
int x;
cin >> x;
 candidates[x - 1]++;
}
for(int i = 0; i < 40; i++)
{
cout << i + 1 << ": " << candidates[i] * 1.0 / N * 100 << "% " << endl;
}
}
При выводе номера кандидата мы обратно возвращаем единицу к индексу. Остановитесь на этом моменте и убедитесь, что вы понимаете код выше. Если возникают трудности, то посмотрите на решение подобной задачи с двумя кандидатами в прошлой статье, перечитайте эту статью и вернитесь к этому моменту. Вы должны понимать, как массив заменил переменные.

Если вы все поняли, то давайте еще немного доработаем программу. Пусть вам на вход теперь подается число 1 <= M <= 100 - число кандидатов. К сожалению нельзя задавать размер массива с помощью переменной. Скоро мы изучим динамические массивы в которых можно изменять размер, но на текущий момент мы этого не умеем. Так что размер массива стоит задать достаточно большим, чтобы он всегда вмещал достаточно информации. Просто лишние переменные мы не будем использовать. Но нельзя создавать слишком большие массивы, ведь они хранятся в вашей оперативной памяти, которая ограниченна. Но обычно, массивы больше миллиона элементов не требуются, а это число легко помещается в память.

Из условий задачи мы видим, что кандидатов никогда не будет больше, чем 100. Тогда давайте перепишем наш код следующим образом.

#include <iostream>

using namespace std;

int main()
{
cout.precision(2);
cout << fixed;
int M;
cin >> M;
int candidates[100];
for(int i = 0; i < M; i++)
{
candidates[i] = 0;
}
int N;
cin >> N;
for(int i = 0; i < N; i++)
{
int x;
cin >> x;
 candidates[x - 1]++;
}
for(int i = 0; i < M; i++)
{
cout << i + 1 << ": " << candidates[i] * 1.0 / N * 100 << "% " << endl;
}
}
Изменилось не много, теперь в циклах мы идем не до 40, а до значения M. Ну и давайте еще немного доработаем эту программу. Пусть теперь сразу после числа M нам даются M имен и фамилий кандидатов, каждый с новой строки. И в финальных списах мы будем выводить не их номер, а их имя и фамилию. Для этого создадим массив строк. Стоит учесть, что когда вы считываете строку, то она считывается только до пробела. Чтобы считать строку полностью, до переноса строки, нужно воспользоваться функцией getline(cin, s);, где s - название строковой переменной. Одна проблема использования этой функции в том, что если вы читаете не с первой строки, то скорее всего курсор находится в конце прошлой строки. В данном случае это так, вы считываете число M и курсор расположен сразу после него. Поэтому нужно считать одну пустую строку перед считыванием имен.
#include <iostream>
#include <string>

using namespace std;

int main()
{
cout.precision(2);
cout << fixed;
int M;
cin >> M;
string names[100], a;
int candidates[100];
getline(cin, a);
for(int i = 0; i < M; i++)
{
candidates[i] = 0;
getline(cin, names[i]);
}
int N;
cin >> N;
for(int i = 0; i < N; i++)
{
int x;
cin >> x;
 candidates[x - 1]++;
}
for(int i = 0; i < M; i++)
{
cout << names[i] << ": " << candidates[i] * 1.0 / N * 100 << "% " << endl;
}
}

Строки это тоже массивы
На самом деле string представляет из себя массив типа char. Поэтому можно, например, получить пятый символ в строке s[4]. Символы в строке можно также изменять как и в массиве. Например, этот код выведет HelXo.
string a = "Hello";
a[3] = 'X';
cout << a;
Интересный факт, который поможет вам в будущем. Пусть у вас есть символ x. Из-за того, что все символы кодируются в системе ASCII и все цифры и буквы находятся рядом друг с другом в порядке возрастания, то:
Если x - цифра, то x - '0' это численное значение этой цифры.
Если x - заглавная буква, то x - 'A' + 1 это ее номер в алфавите.
Если x - прописная буква, то x - 'a' + 1 это ее номер в алфавите.
Тогда код находящий сумму цифр числа может выглядеть так:
string a = "15782";
int ans = 0;
for (int i = 0; i < a.size(); i++)
{
ans += a[i] - '0';
}
cout << ans;
Из этого кода вы также можете заметить новую функцию a.size(). Эта функция возвращает размер строки. К сожалению у массива похожей функции нет.
Многомерные массивы
Вы можете легко создать двухмерный массив (табличка) просто добавив еще одни квадратные скобки к массиву.
int a[2][2] = {{0, 1}, {3, -1}};
Так как двухмерный массив это по сути массив массивов чисел, то инициализация будет выглядеть именно так. Получать элементы можно также, как и в одномерном массиве, только используя вторые квадратные скобки. a[1][0] - там будет лежать тройка. Вы также можете использовать и больше измерений, если захотите.
Инициализировать многомерные массивы можно также с помощью цикла.
int a[15][30][20];
for (int i = 0; i < 15; i++)
{
for (int j = 0; j < 30; j++)
 {
 for (int k = 0; k < 20; k++)
 {
 a[i][j][k] = 0;
}
 }
 }
Может сейчас многомерные массивы не окажутся особо полезными для вас. Но когда вы начнете изучать алгоритмы, которые мы рассказываем в нашем втором курсе, вы поймете насколько это важный элемент программирования.
