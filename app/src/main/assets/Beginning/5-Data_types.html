Типы данных

В прошлом уроке мы познокомились с переменными. Оказалось, что у каждой переменной есть свой тип. Мы изучили тип int - целочисленный тип данных. Но существуют и другие. Что делать, если вам нужно хранить строку или дробное число? Сейчас разберемся.

Чтобы хранить вещественные (числа, поддерживающие цифры после запятой) числа существует тип данных float.

float a = 5.0;
Дробная часть числа записывается через точку. Все операции такие же, как и у int, только поддерживают дробную часть.
Операция деления производится без округления (3.0 / 2.0 = 1.5). Операция остатка от деления невозможна.
Если вы не пишите число через точку, то оно считается за целое. Например
3 / 2 = 1
3.0 / 2.0 = 1.5
Но если хоть одно число дробное, то результат тоже будет дробным. Например.
3.0 / 2 = 1.5

Чтобы указать комплиятору сколько цифр после точки вы хотите выводить используются следующие команды.

cout.precision(3);
cout << fixed;
Команда cout.precision(x) указывает, что нужно выводить x цифр после запятой. Если вы укажете 0, то будут выводится целые числа. Команда cout << fixed запрещает вывод чисел в научной форме (Например, 1,3e-10).
Например, вот полный код программы, которая читает два дробных числа и выводит их сумму, разность, произведение и частное.
#include <iostream>

using namespace std;

int main()
{
cout.precision(3);
cout << fixed;
double a, b;
 cout << "Enter two fractional numbers" << endl;
 cin >> a >> b;
 cout << "The sum is " << a + b << endl;
 cout << "The difference is " << a - b << endl;
 cout << "The product is " << a * b << endl;
 cout << "The quotient is " << a / b << endl;
 }

Почему же тогда всегда не использовать float? Есть несколько проблем связанных с хранением чисел в памяти компьютера.

int и float занимают по 4 байта (32 бита) в памяти компьютера. Это означает, что они могут хранить лишь 2^32 различных чисел. Это 9 - 10 знаков в десятичной системе счисления. Что это значит? Тип int может хранить только числа в пределах двух миллиардов, если вы будете использовать числа больше, программа не будет работать правильно. float - тип с плавающей точкой. Это означает, что он хранит то же самое целое число и степень двойки, на которую нужно домножить. Давайте рассмотрим пример.
У вас есть число 2,25. В двоичной системе это 10,01. Значит тип float хранит число 1001 и степень двойки -2.
Если вы не поняли этого, то можете представлять себе это так, компьютер хранит целое число и место куда нужно поставить точку. Например 225 и 2 (точку поставить перед вторым знаком с конца). Проблема в том, что чем больше у вас целая часть, тем меньше знаков вы сможете хранить в дробной. Например если у вас есть число 99999,5874613, то компьютер обрежет его и точно будет известно только это 99999,587. Остальное же будет заполнено случайными цифрами. Из-за этого возникает погрешность в вычислениях, что может привести к неправильному результату. Но засчет плавающей точки float позволяет хранить довольно большие числа. Правда точными там будут только первые несколько знаков. Кроме этого есть еще и проблема с переодичными дробями. Например число 0,1 нельзя точно представить в двоичной системе, так как оно там переодично. Значит после вывода на экран вы можете увидеть 0,0(9) или 0,10000(1). Тип int лишен такой проблемы и хранит все числа в диапозоне от -2^31 до 2^31 абсолютно точно.
Главное из "дополнительного чтения" сверху. Старайтесь использовать целочисленные типы данных там, где это возможно, так как дробные могут давать погрешность. Тип int занимает 4 байта и может хранить только числа от -2147483648 до 2147483648.

Как же быть, если нужно использовать большие или более точные числа? Первое решение использовать другие типы данных.

На практике обычно используют int, а в случае недостатка размера чисел используют long long жертвуя памятью. Для дробных чисел обычно используют double.
Обычно long long (18 знаков) хватает, чтобы хранить любые числа, возможные в вашей программе. Но если вдруг вам нужны огромные числа, то можно использовать алгоритмы длинной арифметики. Мы расскажем о них в продвинутом курсе.
Нельзя создавать переменные с разными типами данных в одной команде. Правильно делать так.
string a, b, c;
int d, e, f;
Также нельзя давать переменным имена совпадающие с названиями типов данных, или каких либо функций, которые вы подключили.
То есть запрещено писать так. int cout; int int;

Существует логический тип bool, который может хранить только значения 0 или 1 (правда или ложь). Это очень нужный тип и скоро вы сами убедитесь в этом.

Строковые типы данных
Чтобы использовать строки в C++ нужно подключить библиотеку string.
#include <string>
Теперь вы можете использовать тип string. Например
string a = "Hello world!";
Или
#include <iostream>
#include <string>

using namespace std;

int main()
{
string a;
 cout << "What is your name?" << endl;
 cin >> a;
 cout << "Hi, " << a << "!";
 }
Также вы можете использовать тип char, хранящий один символ. Для использования этого типа не нужно подключать библиотеки.
char a = 'T';
Обратите внимание, что строки хранятся в двойных кавычках, а символы в одинарных.
Давайте повторим весь пройденный материал написав программу-анкету.
#include <iostream>
#include <string>

using namespace std;

int main()
{
cout.precision(1);
 cout << fixed;
 string name, country;
 int age, favorite, countFriends;
cout << "What is your name?" << endl;
 cin >> name;
 cout << "Where are you live?" << endl;
 cin >> country;
 cout << "How old are you?" << endl;
 cin >> age;
 cout << "What is your favorite number?" << endl;
 cin >> favorite;
 cout << "How many friends do you invite to your party?" << endl;
 cin >> countFriends;
 cout << "Hi, " << name << " from " << country << "!" << endl;
 cout << "Sum of your age and favorite number is " << age + favorite << endl;
 cout << "If you have one cake at the party, everyone will get " << 1.0 / (countFriends + 1) * 100 << "% from it" << endl;
 }
