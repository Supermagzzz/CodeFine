Простая программа
Код из Codeblocks:

#include <iostream>

using namespace std;

int main()
{
    cout << "Hello world!" << endl;
    return 0;
}

Давайте разберем программу, которую вы увидели при открытии Codeblocks, и посмотрим как она работает.

Пока вы можете считать, что весь код программы будет находиться внутри фигурных скобок после main()
(Скоро вы узнаете, что это немного не так).

int main()
{
    Ваш код будет находиться тут.
}
Значит, за работу программы отвечают следующие две строчки.
    cout << "Hello world!" << endl;
    return 0;
Первая из них выводит надпись "Hello world!" на экран.
cout (Console Out) - Вывод в консоль.
Стрелочками "<<"" вы отправляете "Hello world!" на вывод в консоль.
endl - это символ переноса строки.
Каждая строка программы должна заканчиваться символом ; . Так компилятор понимает, где кончается строка.
Давайте немного поэкспериментируем. Напишите свое имя вместо "Hello world!". Учтите, что весь текст в языке C++ должен находиться внутри двойных кавычек. Советуем пока пользоваться только английским алфавитом, так как другие языки не обязательно будут корректно отображаться. Позже мы покажем, как это исправить.
Ваша программа может выглядеть так.
#include <iostream>

using namespace std;

int main()
{
    cout << "Maxim" << endl;
    return 0;
}

Данные выводятся на экран в том порядке, в котором вы передали их с помощью символов "<<".
Например, прошлую программу можно было написать так.

#include <iostream>

using namespace std;

int main()
{
    cout << "M" << "a" << "x" << "i" <<"m" << endl;
    return 0;
}
Теперь попробуем использовать "endl" (символ переноса строки). Все, что вы будете выводить после него, будет отображаться на следующей строке. Чтобы понять лучше, запустите следующий код.
#include <iostream>

using namespace std;

int main()
{
    cout << "Line 1" << endl;
    cout << "Line 2" << endl;
    cout << "Line 3";
    return 0;
}
Да, необязательно выводить всю информацию за один раз. Можно использовать "cout" в любом месте, сколько угодно раз. Например, можно вывести имя ещё и таким способом.
#include <iostream>

using namespace std;

int main()
{
    cout << "M";
    cout << "a";
    cout << "x";
    cout << "i";
    cout << "m";
    return 0;
}
А что же значит "return 0;"?. Это означает конец вашей программы. Весь код после него не будет выполняться.
#include <iostream>

using namespace std;

int main()
{
    cout << "Line 1";
    return 0;
    cout << "Line 2";
}
На экране будет выведено только "Line 1".
Но если вы хотите, чтобы весь код вашей программы выполнился, то необязательно писать "return 0;". Такой код, тоже правильный.
#include <iostream>

using namespace std;

int main()
{
    cout << "Hello world!" << endl;
}
Мы разобрались, как работает программа из Codeblocks, которая выводила "Hello world!" на экран. Давайте теперь разберемся, что же делают остальные строчки.
#include <iostream>

using namespace std;
Изначально в языке C++ очень мало возможностей. Это сделано для того, чтобы ваши программы были быстрыми и занимали мало места. Если вам нужна какая-то дополнительная функциональность, ее можно подключить с помощью библиотек.
Библиотека "iostream" нужна, чтобы подключить возможность ввода и вывода данных в консоль.
iostream - Input Output STREAM (Поток ввода и вывода).
Без подключения этой библиотеки C++ не будет знать, что такое "cout" и "endl".
Чтобы подключить библиотеку, используется команда:
#include <Название вашей библиотеки>
using namespace std; - это пространство имен. Позже мы расскажем, что это такое, но сейчас можно понимать эту команду так.
Все функции, которые вы дополнительно подключаете, имеют приставку "std::". И команда "using namespace std;" убирает эту приставку. Но вам никто не запрещает писать и так:
#include <iostream>

int main()
{
    std::cout << "Hello world!" << std::endl;
    return 0;
}
У этих двух подходов есть свои плюсы и минусы на практике. Но, для обучения, убрать приставку будет std:: удобнее. Мы советуем делать так при решении наших задач.
