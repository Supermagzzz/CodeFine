Простая программа
Код из Codeblocks:

#include <iostream>

using namespace std;

int main()
{
    cout << "Hello world!" << endl;
    return 0;
}

Давайте разберем программу, которую вы увидели при открытии Codeblocks, и посмотрим как она работает.

Пока что вы можете считать, что весь код программы будет находиться внутри фигурных скобок после main()
(Скоро вы узнаете, что это немного не так)

int main()
{
    Ваш код будет находиться тут.
}
Значит, за работу программы отвечают следующие две строчки.
    cout << "Hello world!" << endl;
    return 0;
Первая из них выводит надпись "Hello world!" на экран.
cout произошло от слов (Console Out - Вывод в консоль)
Стрелочками << вы как-будто отправляете на вывод в консоль "Hello world!", а следом endl.
endl - это символ переноса строки.
Давайте немного поэкспреминтируем. Напишите свое имя вместо "Hello world!". Учтите, что весь текст в языке C++ олжен находиться внутри двойных кавычек. Советуем пока пользоваться только английским алфавитом, так как другие языки не обязательно будут корректно отображаться. Позже мы покажем, как это исправить.
Каждая строка программы должна заканчиваться символом ; . Так компилятор понимает, где кончается строка.
Ваша программа может выглядеть так.
#include <iostream>

using namespace std;

int main()
{
    cout << "Ivan" << endl;
    return 0;
}

Данные выводятся на экран в том порядке, в котором вы передали их с помощью символов <<.
Например, прошлую программу можно было написать так.

#include <iostream>

using namespace std;

int main()
{
    cout << "I" << "v" << "a" << "n" << endl;
    return 0;
}
endl - это символ переноса строки. После него все, что вы будете выводить будет отображаться на следующей строке. Чтобы понять лучше, запустите следующий код.
#include <iostream>

using namespace std;

int main()
{
    cout << "Line 1" << endl;
    cout << "Line 2" << endl;
    cout << "Line 3";
    return 0;
}
Да, необязательно выводить всю информацию за один раз. Можно использовать cout в любом месте, сколько угодно раз. Например, можно вывести имя ещё и таким способом.
#include <iostream>

using namespace std;

int main()
{
    cout << "I";
    cout << "v";
    cout << "a";
    cout << "n";
    return 0;
}
А что же значит return 0; ?. Это означает конец вашей программы. Весь код после него не будет выполняться.
#include <iostream>

using namespace std;

int main()
{
    cout << "Line 1";
    return 0;
    cout << "Line 2";
}
На экране будет выведено только Line 1.
Но если вы хотите, чтобы весь код вашей программы выполнился, то необязательно писать return 0. Такой код, тоже правильный.
#include <iostream>

using namespace std;

int main()
{
    cout << "Hello world!" << endl;
}
Мы разобрались, как работает программа из Codeblocks, которая выводила Hello world! на экран. Давайте теперь разберемся, что же делают оостальные строчки.
#include <iostream>

using namespace std;
Изначально в языке C++ очень мало возможностей. Это сделано для того, чтобы ваши программы были быстрыми и занимали мало места. Если вам нужна какая-то дополнительная функциональность, ее можно подключить с помошью библиотек.
Библиотека iostream нужна, чтобы подключить возможность ввода и вывода данных в консоль.
iostream - Input Output STREAM (Поток ввода и вывода).
Без подключения этой библиотеки C++ не будет знать, что такое cout и endl.
Чтобы поключить библиотеку используется команда:
#include <Название вашей библиотеки>
using namespace std; - это пространство имен. Позже мы расскажем, что это такое, но сейчас можно понимать эту команду так.
Все функции которые вы дополнительно подключаете имеют приставку std:: . Это команда убирает эту приставку. Но вы можете писать так:
#include <iostream>

int main()
{
    std::cout << "Hello world!" << std::endl;
    return 0;
}
У этих двух подходов есть свои плюсы и минусы на практике. Но для обучения убрать приставку std:: удобнее. Мы советуем делать так при решении наших задач.
