Первая программа
Код из Codeblocks:

#include <iostream>

using namespace std;

int main()
{
    cout << "Hello world!" << endl;
    return 0;
}

Разберем программу, которую вы увидите при открытии Codeblocks.

Считайте, что весь код программы находится между фигурными скобками после main()
(Это немного не так, полную информацию вы узнаете позже).

int main()
{
    Ваш код будет находиться тут.
}
За работу программы отвечают две следующие строки.
    cout << "Hello world!" << endl;
    return 0;
Первая из них выведет на экран надпись "Hello world!".
cout (Console Out) - Вывод в консоль.
Знаком << вы отправите "Hello world!" на вывод в консоль.
endl - это символ переноса строки.
Каждая строка программы должна заканчиваться символом ; . Так компилятор поймет, где закончилась строка.

Напишите свое имя вместо "Hello world!". Учтите, что весь текст в языке C++ должен находиться внутри двойных кавычек, воспльзуйтесь английским алфавитом, так как другие языки не корректно отобразятся (Позже мы покажем, как это исправить).
Пример:
#include <iostream>

using namespace std;

int main()
{
    cout << "Maxim" << endl;
    return 0;
}

Данные выведутся на экран в том порядке, в котором вы передадите их с помощью символа <<.
Например, прошлую программу можно было написать так:

#include <iostream>

using namespace std;

int main()
{
    cout << "M" << "a" << "x" << "i" <<"m" << endl;
    return 0;
}
Теперь используем endl (символ переноса строки). Все, что вы выведите после него, отобразится на следующей строке. Чтобы понять лучше, запустите следующий код.
#include <iostream>

using namespace std;

int main()
{
    cout << "Line 1" << endl;
    cout << "Line 2" << endl;
    cout << "Line 3";
    return 0;
}
Да, необязательно выводить всю информацию за один раз. Можно использовать cout в любом месте, сколько угодно раз. Например, можно вывести имя ещё и таким способом.
#include <iostream>

using namespace std;

int main()
{
    cout << "M";
    cout << "a";
    cout << "x";
    cout << "i";
    cout << "m";
    return 0;
}
А что же значит return 0; ? Это означает конец вашей программы. Весь код после него не выполнится.
#include <iostream>

using namespace std;

int main()
{
    cout << "Line 1";
    return 0;
    cout << "Line 2";
}
На экране будет выведено только "Line 1 ".
Если вы хотите, чтобы весь код, вашей программы, выполнился, то необязательно писать return 0; . Такой код, также будет являться правильным.
#include <iostream>

using namespace std;

int main()
{
    cout << "Hello world!" << endl;
}

Мы разобрались, как работает программа из Codeblocks, которая выводила "Hello world!" на экран. Теперь посмотрим, что делают остальные строки.
#include <iostream>

using namespace std;
Изначально в языке C++ очень мало возможностей. Это сделано для того, чтобы ваши программы были быстрыми и занимали мало места. Если вам нужна какая-то дополнительная функциональность, ее можно подключить с помощью библиотек.

Библиотека iostream требуется для возможности использования ввода и вывода данных в консоль.
iostream - Input Output STREAM (Поток ввода и вывода).
Без подключения этой библиотеки C++ не будет знать, что такое cout и endl.

Для подключения библиотеки, используйте команду:
#include <Название вашей библиотеки>
using namespace std; - это пространство имен. Позже мы расскажем, что это такое, но сейчас можно понимать эту команду так.
Все функции, которые вы дополнительно подключете, требуют приставку std:: , а команда using namespace std; убирет эту приставку. Вам никто не запретит писать и так:
#include <iostream>

int main()
{
    std::cout << "Hello world!" << std::endl;
    return 0;
}
У этих двух подходов есть свои плюсы и минусы на практике. Но, для обучения, убрать приставку std:: будет удобнее. Мы советуем делать так при решении наших задач.
