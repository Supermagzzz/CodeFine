Переменные

В прошлом уроке мы научились выводить данные на консоль. Но как можно ввести данные в программу?
В этом уроке мы попробуем написать свой простой консольный калькулятор, который умеет складывать, вычитать и умножать два числа.

Если вы считаете данные с экрана их нужно будет где-то хранить. Для этого у копьютера есть оперативная память.
Что бы хранить данные в памяти, нужно создавать для них переменные. Переменные создаются следующим образом.

#include <iostream>

using namespace std;

int main()
{
    тип переменной название = число;
}
У каждой переменной есть свой тип. Про то, какие типы существуют мы поговорим чуть позже, а сейчас будем работать с целчм числовым типом.
Чтобы хранить в памяти целое число нужно использовать тип int. Название переменной вы можете придумать сами, оно нужно чтобы обращаться к ней. То есть, если у вас есть переменная с именем a, то каждый раз, когда вы будете писать в вашей программе слово a, он будет подставлять вместо него значение вашей переменной. Чтобы было понятнее, посмотрим пример.
#include <iostream>

using namespace std;

int main()
{
    int a = 5;
    cout << a;
}
Этот код выведете на экран число 5. Учтите, что a - это не текст, а переменная и ее нельзя писать в кавычках. Есть несколько правил, по которым вы можете придумывать имена для переменных. Они должны состоять из английских букв, цифр и символа подчеркивания. Цифра не может быть первым символом в имени переменной. Нельзя давать переменным имена, которые совпадают с другими названиями в C++. Например запрещено давать такие имена, как int, cout, endl и т.д.

Можно создавать несколько переменных.

#include <iostream>

using namespace std;

int main()
{
    int a = 5;
    int b = 7;
    cout << a << " " << b;
}
Этот код выведет на экран сначала первую переменную, потом пробел и потом вторую переменную. В итоге на экране будет
5 7
Как получить значение переменной с экрана? Для этого воспользуемся cin - (Console in) вводом с консоли. Это будет выглядеть также как и cout, но только стрелочки будут в другую сторону.
#include <iostream>

using namespace std;

int main()
{
    int a = 0;
    cin >> a;
    cout << a;
}
Если вы запустите эту программу, то она сначала ничего не выведет и будет ждать вашего ввода. Введите какое-нибудь число и нажмите Enter. Программа прочитает его и запишет в переменную, а потом выведет ее на экран. Зачем записывать в переменную 0, если прямо сейчас консоль запишет в нее другое значение? Да, можно ничего не записывать в переменную.
#include <iostream>

using namespace std;

int main()
{
    int a;
    cin >> a;
    cout << a;
}
Этот код тоже работает. А еще можно объявлять несколько переменных одного типа в одной строке через запятую. Учтите, что тип переменной при этом пишется только один раз в начале строки. При вводе нескольких чисел в консоль их нужно разделять пробелом. Запустите следующую программу и введите два числа через пробел. Программа напечатает эти два числа и число 25.
#include <iostream>

using namespace std;

int main()
{
    int a, b, c = 25;
    cin >> a >> b;
    cout << a << " " << b << " " << c;
}
К числовым переменным можно применять математические операции. В переменную можно записывать не только число, но и математическое выражение.
#include <iostream>

using namespace std;

int main()
{
    int a = 5 + 5 * 7;
    int b = a + a, c;
    cin >> c;
    b = b + c;
    cout << a << " " << b << " " << c;
}
Тут стоить учесть еще одно правило. Тип переменной мы указываем, только при ее создание (т.е при первом объявлении). Если вы в дальнейшем перезаписываете переменную, то тип указывать нельзя.
Давайте разберемя, что делает программа.
int a = 5 + 5 * 7;
Объявили переменную a равную 40.
int b = a + a;
Объявили переменную b равную 80 (40 + 40).
int c;
cin >> c;
Считали переменную c с консоли. (Предположим, что мы ввели число 20).
b = b + c;
Прибавили к переменной b число 20, теперь оно равно 100 (b = 80 + 20).
Вывели все три числа на экран. В консоли будет находится 40 100 20.

Мы достаточно быстро разобрали основы работы с переменными, но это очень важная тема. Остановитесь здесь, и убедитесь, что вы понимаете, как работает программа выше и можете написать ее сами. Если возникают трудности, советуем перечитать статью.

Математические операции можно применять и при выводе данных на экран. Давайте напишем программу, которая читает два числа и выводит их сумму двумя способами.

Способ 1

#include <iostream>

using namespace std;

int main()
{
    int a;
    int b;
    cin >> a >> b;
    int sum = a + b;
    cout << sum;
}
Способ 2
#include <iostream>

using namespace std;

int main()
{
    int a, b;
    cin >> a >> b;
    cout << a + b;
}
Оба способа работают одинаково, можете убедиться в этом. Давайте теперь рассмотрим основные арифмитические операции с целыми числами.
с = a + b; Операция сложения двух чисел.
с = a - b; Операция вычитания двух чисел.
с = a * b; Операция умножения двух чисел.
С делением все не так просто. Так как результат деления может быть не целым, то мы не сможем записать его в целый тип данных. В следующем уроке мы рассмотрим решение этой проблемы и узнаем про другие типы данных. Но у типа int есть следующие операции.
с = a / b; Целая часть от деления a на b (Пример 10 / 3 = 3, 15 / 3 = 5).
с = a % b; Остаток от деления a на b (Пример 10 % 3 = 1, 15 % 3 = 0).
Если вы хотите прибавить к переменной число или применить к ней любую другую операцию, то не обязательно писать так:
a = a + b;
Вместо этого можно воспользоваться операцией прибавления:
a += b; (Если a было равно 5, а b было равно 3, то a станет равно 8).
Также можно делать с любой операцией. Примеры:
a -= 2;
a *= b;
a /= b - c + 35 * 60;
Если вы хотите прибавть или отнять единицу от числа, то можно воспользоваться следующей формой.
a++; (Прибавляет 1 к a)
a--; (Отнимает 1 от a)

Обладая всеми этими знаниями нам не составит труда написать свой калькулятор. Давайте заодно научимся решать наши упражнения, где нужно считывать данные.
Вот вам задача:

На вход подаются два целых числа. Выведите их сумму, разность и произведение, каждое число должно быть с новой строки.

Входные данные
5 3
Выходные данные
8
2
15
Сначала идет условие задачи, потом пример тестов, чтобы показать вам, как должна работать ваша программа. Исходя из входных данных вы должны считать два числа. Исходя их выходных вывести три числа, каждое в новой строке. Давайте напишем такую программу.
#include <iostream>

using namespace std;

int main()
{
    int a, b;
    cin >> a >> b;
    cout << a + b << endl;
    cout << a - b << endl;
    cout << a * b << endl;
}
Вот так нужно оформлять подобные задачи. Напоследок, давайте добавим поясняющий текст к нашей программе, чтобы пользователю было удобнее использовать ваш калькулятор. (так вы можете делать в своих личных программах, но при решении упражнений придерживайтесь четкого формата данных. Не выводите ничего лишнего. То что выведет ваша программа на ввод из Входных данных должно полностью совпадать с тем, что находится в Выходных данных)
#include <iostream>

using namespace std;

int main()
{
cout.precision(3);
cout << fixed;
int a, b;
 cout << "Enter two fractional numbers" << endl;
 cin >> a >> b;
 cout << "The sum is " << a + b << endl;
 cout << "The difference is " << a - b << endl;
 cout << "The product is " << a * b << endl;
 }
