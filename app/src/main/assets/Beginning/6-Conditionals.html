Условный оператор

До сих пор наши программы всегда шли по одному четкому сценарию. Но зачастую хочется, чтобы программа меняла свое поведение в зависимости от разных событий. Предположим, что вы знаете возраст пользователя и, используя это, хотите показать ему определенную информацию. Что делать в таких случаях?

Для этого нам и понадобится тип bool. Напомним, это тип данных, который принимает одно их двух значений true(правда) или false(ложь).

Предположим у вас есть переменная age в которой находится возраст пользователя. Вы хотите проверить есть ли пользователю сорок лет.

bool a = (age >= 40);
Если выполнится условие age >= 40, то в переменную a запишется true, в противном случае false.
Задавать такие вопросы вы можете с помощью следующих операторов.
> - больше.
< - меньше.
>= - больше либо равно.
<= - меньше либо равно.
== - равно.
Обратите внимание, что для проверки равенства используется два знака равно подряд. Это так, потому что один знак равно используется для присваивания значения переменной. Это частая ошибка, будьте внимательнее и не путайте эти два оператора.

Но что делать с этим значением?
Существует контрукция if (условие) {код}, где код внутри фигурных скобок будет выполнять только если условие равно true.

bool a = (age >= 10);
 if (a)
{
cout << "A";
 }
cout << "B";
Если пользователю меньше 10 лет, то он увидит "B", а все остальные увидят "AB".
Условие про возраст накладывается только на команду cout << "A";. Так как cout << "B"; не лежит внутри фигурных скобок условия, то эта команда выполнится в любом случае. Код ограниченный фигурными скобками называется "Блок кода". Если один блок кода вложен в другой, то для кода внутри него принято делать отступ в один таб или четыре пробела. Так легко визуально понять какой блок кода чему принадлежит. Изучите полный код программы для лучшего понимания.
#include <iostream>

using namespace std;

int main()
{
int age = 24;
bool a = (age >= 10);
 if (a)
{
cout << "A";
 }
cout << "B";
 }
На самом деле, на практике никто не создает отдельную переменную, чтобы хранить результат условия. Можно писать условие сразу внутри скобок if().
if (age >= 10)
{
cout << "A";
 }
cout << "B";
Блоки кода if можно вкладывать друг в друга.
if (age >= 18)
{
if (country == "Russia")
 {
cout << "Hi, adult from Russia";
 }
 if (country == "America")
 {
cout << "Hi, adult from America";
 }
 }
Значит, людям которым больше 18 лет и которые живут в Америке или в России выведется сообщение, а всем остальным нет. Существует также конструкция else(иначе). Синтаксис следующий
if(условие)
{код}
else
{код}
Блок кода внутри else выполнится, если не выполнится блок кода внутри if. Например так может выглядеть код, проверяющий можете ли вы купить товар в интернет-магазине.
if (money >= 100)
{
cout << "You can buy this item";
}
else
{
cout << "You don't have enough money";
}
Также вы можете пользоваться конструкцией else if(условие) {код}. Таким образом вы можете собирать целые цепочки условий.
if (a >= 100)
{
cout << "A";
}
else if (a >= 80)
{
cout << "B";
}
else if (a >= 60)
{
cout << "C";
}
else
{
cout << "D";
}
Первым в цепочке условий обязан быть блок if. Дальше может быть сколько угодно блоков else if(может не быть вообще). И можно добавить блок else последним. Тогда условия будут проверяться сверху вниз и выполнится то, которое подходит первым и проверка условий закончится, а код продолжит выполняться с первой строки после всей текущей цепочки условий. Но если нет ни одного подходящего условия, но существует блок else, то выполнится он.
Например, если a = 85, то прошлый код выведет B. В отличии от кода ниже, который выведет BC.
if (a >= 100)
{
cout << "A";
}
if (a >= 80)
{
cout << "B";
}
if (a >= 60)
{
cout << "C";
}
Так как в этом коде все блоки if это отдельные цепочки условий, то произойдет проверка всех условий.

Сложные условия

Часто нужно писать сложные условия, состоящие из нескольких простых проверок, которые мы делали раньше. Например
Если количество денег больше либо равно стоимости товара + стоимость доставки и название страны равно "Россия" и возраст больше 18.
Чтобы комбинировать условия, используются операторы (&& - "И") и (|| - "ИЛИ").
Например пусть нам нужно условие, которое будет проверять, что человек находится в возрасте от 18 до 30 лет. Но оператора от ... до ... у нас не существует. Давайте распишем это условие с помощью уже известных нам операторов. Возраст должен быть больше либо равен 18 и меньше либо равен 30. Напишем это.

if (age >= 18 && age <= 30)
Оператор && соединяет два условия словом И. Если вы помните алегбру логики из школьных уроков информатики, то вам будет проще. Можно также пользоваться скобками. Например
(условие1 && (условие2 || условие3)) Тогда сначала применится операция || между вторым и третьим условием, а потом операция && между первым условием и прошлым результатом.
Чтобы немного формализовать давайте посмотрим на следующую табличку.

Лучшее понимание сложных условий к вам придет с опытом. А сейчас давайте выполним задачу, которую поставили себе в начале этой части статьи.
Если количество денег больше либо равно стоимости товара + стоимость доставки и название страны равно "Россия" и возраст больше 18.
if (money >= itemCost + deliveryCost && country == "Russia" && age > 18)
В C++ у каждой операции существует приоритет. Например, то что в скобках будет выполняться раньше. Умножение выполняется раньше сложения. Все как в математике. Операции "больше", "равно", "И" и другие, которые мы изучили в этой статье выполняются позже всех математических операций. Операции "И" и "ИЛИ" выполняются позже всех, это позволяет вам писать ваши условия без скобок. Но если вы понимаете, что в вашей программе есть баг, то рекомендуем проверить правильность расстановки скобок в сложных условиях.
